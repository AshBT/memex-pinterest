The current process of finding relevant human trafficking websites is purely manual (clicking around), this tool aims to assist in creating a simple pluggable system that can find potentially relevant websites.

Our simple analysis system is basic, but we are hoping somewhat effective.

There are two ways to use this tool:

  • A basic analysis of keywords in Common Crawl organized by potential relevance
  • A directed crawler/scraper that is able to crawl websites linked to sites entered by a user
